## 24.0.4-03 — Reset Sound (`res/raw/db_wipe.ogg`)

Import the demolition sound effect for the Reset App function (spec 0.4; played via `SoundPool`, spec §5). The `.ogg` was synthesized in the spec session — no license required.

### Steps

```zsh
mkdir -p ~/AppDevelopment/icodeforyou_android/CatsCarsCoins/app/src/main/res/raw
cp <path-to-your>/db_wipe.ogg ~/AppDevelopment/icodeforyou_android/CatsCarsCoins/app/src/main/res/raw/db_wipe.ogg
```

Rules: `res/raw/` filenames are lowercase `a-z 0-9 _` only, and the name `db_wipe` is load-bearing — code will reference it as `R.raw.db_wipe`. No transcoding needed; OGG is natively supported by `SoundPool` on minSdk 31.

### Verify

`res/raw/db_wipe.ogg` appears in the Project view; double-click plays it in the IDE's audio preview.
