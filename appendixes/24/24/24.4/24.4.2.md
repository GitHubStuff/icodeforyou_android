## Design (parallel to `CatsRepository`)

1. **One observable, query-driven.** `manufacturers(query)` — blank →
   all, non-blank → FTS4 over the two indexed fields named in 24.4.1.
2. **`refresh()` upserts by id**, idempotent on `Mfr_ID`; throws on
   failure (presentation belongs to callers).
3. **`FETCH_PAGE = 1`** on the companion — vPIC `GetAllManufacturers` is
   paginated (~100/page); the demo fetches the first page. (A "load more"
   is out of scope; YAGNI unless the spec's acceptance asks for it.)
4. **`clearAll()`** — the Reset App hook, uniform with Cats and Coins.
5. Pure domain: sole import is `Flow`.

```kotlin
// cars/domain/CarsRepository.kt
// CatsCarsCoins — spec 24.4.2. Complete file.
package com.icodeforyou.catscarscoins.cars.domain

import kotlinx.coroutines.flow.Flow

/**
 * Domain contract for stored manufacturers (spec Phase 4: NHTSA vPIC +
 * Room FTS4 search). Pure Kotlin — the data layer implements over Room +
 * Retrofit; consumers see only this interface (DIP). Parallel to
 * CatsRepository by design.
 */
interface CarsRepository {

    /**
     * The stored manufacturers matching [query], emitting on every change.
     *
     * - Blank [query] → every stored manufacturer.
     * - Non-blank → FTS4 match over the two indexed fields (24.4.1: name,
     *   country).
     *
     * One observable, not a list-val + search pair — the Cars screen is a
     * single list under a single search field.
     */
    fun manufacturers(query: String): Flow<List<Manufacturer>>

    /**
     * Fetches the first page of manufacturers from vPIC and upserts by
     * [Manufacturer.id] — repeated refreshes never duplicate rows. Throws
     * on network or parse failure; failure presentation (spec: read-only
     * toast, 750 ms) is the caller's, not this layer's.
     */
    suspend fun refresh()

    /** Reset App hook (spec §16) — each feature clears its own tables. */
    suspend fun clearAll()

    companion object {

        /** vPIC page fetched by [refresh] (1-indexed, ~100 rows/page). */
        const val FETCH_PAGE: Int = 1
    }
}
```
