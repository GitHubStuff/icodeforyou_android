## Design (parallel to `CatEntity`)

1. **Integer PK.** `Mfr_ID` is the primary key, so the 24.4.2
   upsert-by-id refresh contract is enforced by the schema. Int key, not
   String ŌĆö the one shape difference from cats.
2. **External-content FTS4:** `@Fts4(contentEntity = ManufacturerEntity)`
   on `manufacturers_fts` ŌĆö Room generates the sync triggers, the index
   stores no duplicate text, and the two indexed columns are exactly the
   fields the contract fixed (name, country). Column names match the
   content entity's.
3. Explicit `@ColumnInfo`; CREATE TABLE equivalent in the KDoc; both
   mappers (`toEntity` because refresh writes whole manufacturers).

// cars/data/ManufacturerEntity.kt

package com.icodeforyou.catscarscoins.cars.data

import androidx.room3.ColumnInfo
import androidx.room3.Entity
import androidx.room3.Fts4
import androidx.room3.PrimaryKey
import com.icodeforyou.catscarscoins.cars.domain.Manufacturer

/**
 * Room row for one manufacturer (spec Phase 4). PK is vPIC's Mfr_ID (an
 * Int, unlike the cats table's String key) ŌĆö upsert-by-id (the 24.4.2
 * refresh contract) falls straight out of the primary key.
 *
 * CREATE TABLE equivalent:
 * ```
 * CREATE TABLE manufacturers (
 *     id      INTEGER PRIMARY KEY NOT NULL,
 *     name    TEXT NOT NULL,
 *     country TEXT NOT NULL
 * )
 * ```
 */
@Entity(tableName = "manufacturers")
data class ManufacturerEntity(

    @PrimaryKey
    @ColumnInfo(name = "id")
    val id: Int,

    @ColumnInfo(name = "name")
    val name: String,

    @ColumnInfo(name = "country")
    val country: String,
)

/**
 * FTS4 shadow of [ManufacturerEntity] (spec: Cars search). External-content
 * FTS ŌĆö Room generates the sync triggers; the index stores no duplicate
 * text. Indexes EXACTLY the two searchable fields fixed at 24.4.1: name,
 * country. Column names must match the content entity's.
 */
@Fts4(contentEntity = ManufacturerEntity::class)
@Entity(tableName = "manufacturers_fts")
data class ManufacturerFtsEntity(

    @ColumnInfo(name = "name")
    val name: String,

    @ColumnInfo(name = "country")
    val country: String,
)

fun ManufacturerEntity.toDomain(): Manufacturer = Manufacturer(
    id = id,
    name = name,
    country = country,
)

fun Manufacturer.toEntity(): ManufacturerEntity = ManufacturerEntity(
    id = id,
    name = name,
    country = country,
)
```
