## Design

- **One expression:** `getSpotPrice().data.amount.toUsdCents()` — the wire
  chain from your string-JSON question, completed. DTO shape and string
  amounts are invisible above this file.
- **Failure contract honored by propagation:** Retrofit's network/HTTP
  exceptions and the parser's `NumberFormatException` both escape; the
  engine's tick-skip guard (24.2.23) handles them.
- **No test file — deliberate.** Every part is already under test (parser
  10, engine 11 via the port); the adapter is pure delegation. A test that
  fakes `CoinbaseApi` to assert this one-liner exercises the mock, not the
  code. Delegation-only adapters are the recognized exception to
  test-first; behavior gets covered end-to-end by the phase acceptance run.

package com.icodeforyou.catscarscoins.coins.data.remote

import com.icodeforyou.catscarscoins.coins.domain.CoinPriceSource

/**
 * Data-layer adapter for [CoinPriceSource]: Coinbase spot price →
 * envelope unwrap → exact cents. All wire knowledge (DTO shape, string
 * amounts) ends here; the engine sees only a Long.
 *
 * Failure contract (per the port's KDoc): Retrofit throws on
 * network/HTTP failure and [toUsdCents] throws on a malformed amount —
 * both propagate, and the engine skips that tick.
 */
class CoinbaseCoinPriceSource(
    private val coinbaseApi: CoinbaseApi,
) : CoinPriceSource {

    override suspend fun currentAmountCents(): Long =
        coinbaseApi.getSpotPrice().data.amount.toUsdCents()
}
