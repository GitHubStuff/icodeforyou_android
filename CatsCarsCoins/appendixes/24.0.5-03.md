## 24.0.5-03 — Register `.MyApp` in the Manifest

Replace the entire contents of `app/src/main/AndroidManifest.xml` with the file below. One change from 24.0.4-06: `android:name=".MyApp"` on the `<application>` element — the line deliberately deferred until the class existed (24.0.5-02).

### The file

```xml
<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools">

    <!-- ── The only permissions in the app (spec §4) ── -->
    <!-- Required: all three data sources are network APIs -->
    <uses-permission android:name="android.permission.INTERNET" />
    <!-- Optional per spec: lets polling/refresh gate on connectivity -->
    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />

    <application
        android:name=".MyApp"
        android:allowBackup="true"
        android:dataExtractionRules="@xml/data_extraction_rules"
        android:fullBackupContent="@xml/backup_rules"
        android:icon="@mipmap/ic_launcher"
        android:label="@string/app_name"
        android:roundIcon="@mipmap/ic_launcher_round"
        android:supportsRtl="true"
        android:theme="@style/Theme.CatsCarsCoins">
        <activity
            android:name=".MainActivity"
            android:exported="true"
            android:label="@string/app_name"
            android:theme="@style/Theme.CatsCarsCoins"
            android:windowSoftInputMode="adjustResize">
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />

                <category android:name="android.intent.category.LAUNCHER" />
            </intent-filter>
        </activity>
    </application>

</manifest>
```

**Why:** with this line the OS instantiates `MyApp` at process start, so Koin is running before any Activity exists — the spec's 0.5 wiring is now live end to end (manifest → MyApp → startKoin → empty appModules).

### Verify

```zsh
./gradlew assembleDebug
```
