## 24.0.4-02 — Splash Assets (`splash_logo` + `splash_black`)

Two resources this step (spec 0.4): the splash image from the `.png`, and the black background drawable. Theme wiring is a later step.

### 1. Import the splash image

Copy the splash `.png` into the project as `splash_logo.png`:

```zsh
mkdir -p ~/AppDevelopment/icodeforyou_android/CatsCarsCoins/app/src/main/res/drawable-nodpi
cp <path-to-your-splash>.png ~/AppDevelopment/icodeforyou_android/CatsCarsCoins/app/src/main/res/drawable-nodpi/splash_logo.png
```

Rules: resource filenames are lowercase `a-z 0-9 _` only — the copy above enforces the name. `drawable-nodpi` prevents density scaling, correct for a single splash bitmap.

Size note: on API 31+ the platform splash renders the icon inside a **circular mask ~240dp** in diameter. Artwork with detail near the corners gets cropped — if that matters, say so before the theme step and the image gets padding treatment instead.

### 2. Create the black background drawable

Create new file `app/src/main/res/drawable/splash_black.xml` (**Project view:** right-click `res/drawable` → New → Drawable Resource File, name `splash_black`), full content:

```xml
<?xml version="1.0" encoding="utf-8"?>
<!-- splash_black.xml — solid black splash background (spec 0.4: OS splash -> black) -->
<shape xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="rectangle">
    <solid android:color="#FF000000" />
</shape>
```

### Verify

Both files exist: `res/drawable-nodpi/splash_logo.png`, `res/drawable/splash_black.xml`. Open `splash_black.xml` — preview pane shows a black rectangle. Open `splash_logo.png` — renders in the editor.
